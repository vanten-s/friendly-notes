____SERVER COMMANDS____

# to write data
w<filename.extension>
<data>

# to read the data
r<filename.extension>

# to create a new file
x<filename.extension>

# to remove a file
f<filename.extension>

# to exit the Server
q

____KEY BINDINGS____

# to save
Left control

# to close both the client and the server 
Right control

# to close only the client 
Escape
